Umi Cloudflare Cache Buster

Hi there! This is a simple tool to clear the cache on any cloudflare site on the Umi network. If you would like any other sites added to this, then please do not hesitate to put it in the development channel on slack.

Select Site to Clear Cache

The following page will give you a response from cloudflare. If you see success: true in any part of the response then it has been successful. If not, then tell Harry about the bug!
